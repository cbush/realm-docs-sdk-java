Wrapper around a Row in Realm Core.IMPORTANT: All access to methods using this class are non-checking. Safety guarantees are given by the annotation processor and Object Store's typed Realm schema validation which is called before the typed API can be used.

For low-level access to Row data where error checking is required, use :ref:`CheckedRow <io_realm_internal_CheckedRow>` .
