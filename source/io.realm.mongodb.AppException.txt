io.realm.mongodb.AppException
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. _io_realm_mongodb_AppException:

*Superclass:*

- **java.lang.RuntimeException (?)**

This class is a wrapper for all errors happening when communicating with a MongoDB Realm app. This include both exceptions and protocol errors. Only is guaranteed to contain a value. If the error was caused by an underlying exception is ``null`` and is set, while if the error was a protocol error is set and is null.
