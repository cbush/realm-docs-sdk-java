io.realm.RealmChangeListener<T>
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. _io_realm_RealmChangeListener:

RealmChangeListener can be registered with a , or to receive a notification about updates.

When registered against a ``Realm`` you'll get notified when a Realm instance has been updated. Register against a ``RealmResults``, ``RealmList``, ``RealmDictionary`` or ``RealmObject`` to only get notified about changes to them.



Realm instances on a thread without an cannot register a RealmChangeListener.

All and will automatically contain their new values when the method is called. Normally this means that it isn't necessary to query again for those objects, but just invalidate any UI elements that are using them. If there is a chance that a object has been been deleted, it can be verified by using .
